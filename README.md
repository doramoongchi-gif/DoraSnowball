v0.9.1 (Build 14) - 데이터 안정성 및 시각적 고도화
--------------------------------------------------
- 데이터 유실 방지 시스템(Migration) 도입:
    * 앱 업데이트 시 기존 데이터가 유지되도록 Database Migration(v9 -> v10) 경로 공식 적용
    * Destructive Migration 옵션 제거로 사용자 데이터 보호 강화
- 실시간 환율 엔진 탑재:
    * 고정 환율(1,350원) 탈피, 네트워크를 통한 야후 파이낸스 실시간 USDKRW 환율 연동
    * 거래 종료 시점의 최신 환율을 반영한 자동 원화 환산 기능 구현
- 축하카드 이미지 공유 혁신:
    * 단순 텍스트 공유에서 고해상도 .png 이미지 파일 공유 시스템으로 업그레이드
    * 화려한 디자인: 티커 크기 확대(44sp), 네온 시안 수익률 그라데이션, 선명한 원화/달러 수익금 표시
    * 레이아웃 최적화: 수익금 위치를 좌상단으로 이동하여 배경 이미지(눈사람) 시야 방해 해결
- 카드 및 UI 사용성 개선:
    * 진행율 계산식 변경: '투자금액' 기준에서 '현재 회차 / 전체 분할수' 기준으로 변경하여 직관성 확보
    * 카드 뒷면 가독성: 전일 대비 종가 등락 색상(빨강/파랑) 적용, 누적개수 0일 때 연녹색 배경 강조
    * PIP 모드 최적화: 세로 길이를 10% 추가 확보(10:23 비율)하여 하단 정보 잘림 현상 완벽 해결
    * 폰트 및 색상: 다크 테마 시인성 확보를 위해 밝은 파랑(0xFF42A5F5) 적용 및 전체 폰트 크기 상향
- 시스템 안정성: PIP 모드 진입 시 추가매수 개수 0개 표시 오류 방지 로직 적용

    * 가변 추가매수 시스템: 팝업에서 추가매수 개수(1~10)를 직접 선택 가능하며, 선택 시 금액/수량 즉시 재계산 및 카드별 영구 저장
    * 스마트 UI: '처음LOC매수' 단계에서는 콤보박스를 자동으로 숨겨 사용자 혼선 방지

    * 완전 투명 배경 구현: 시스템 테마 수정을 통해 뒷배경이 비치는 투명 PIP 창 구현

- 거래 종료 축하 시스템 도입:
    * 스타일리시 축하 팝업: img_congrats 배경 및 힙한 타이포그래피(기울기, 그림자 효과)를 활용한 결과 화면 구현
    * 핵심 요약 제공: 티커, 실현 수익, 최종 수익률, 거래 기간 4가지 핵심 지표 중심의 깔끔한 정보 제공
- 카드 관리 및 기타:
    * 카드 별명 수정 기능 추가 및 즉시 반영
    * 카드 앞면 공유 버튼 텍스트 추가로 시각적 통일감 확보
    * 앱 전체 버전 정보 동기화 및 안정성 강화
