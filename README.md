    * 거래 메모 기능: 거래 등록 시 메모 입력 필드 추가 및 뒷면 상세 리스트에서 확인 가능

v0.9.4 (Build 18) - 홈 화면 전면 개편 및 시각적 애니메이션 도입 (2025-05-23)

- 시각적 완성도 및 애니메이션 강화:
    * 로딩 스켈레톤 UI (Skeleton UI) 적용: 데이터 로딩 중 빈 화면 대신 카드 구조의 애니메이션을 보여주어 체감 속도 및 고급스러움 증대

- RSI 알림 섹션 편의성 개선:
    * 가로 스크롤(LazyRow) 지원: 종목이 많아도 한 줄로 깔끔하게 확인 가능
    * 원터치 상세 정보 연결: RSI 종목 칩 클릭 시 해당 ETF의 상세 설명 팝업 즉시 연결
- 레이아웃 및 기타 최적화:
    * 휴장일 정보 재배치: 화면 최상단 우측에 고정 노출하여 가독성 극대화
    * 환율 챠트 시인성 개선: 챠트 높이 및 여백 조정으로 최고/최저가 라벨(금액, 날짜) 완벽 복구
    * 버튼 디자인 정교화: 더보기 탭의 '카드 1장 가져오기' 버튼에 다크 테마 최적화 Tertiary 색상 적용
    * 리포트 가이드 추가: '총 평가 자산' 계산식에 대한 상세 설명 문구 삽입
